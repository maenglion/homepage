Roadmap & Milestones

우리가 걸어온 길

CASP ENGINE은 “어떻게 말했는가”를 데이터로 만들기 위해, 대화 기반 리포트 → 음성 기반 비언어 지표 분석 → 프로토콜화의 순서로 발전해왔습니다.

회사 설립일: 2025-07-21

Milestones

CASP ENGINE은 신경다양성(ASD/ADHD 등) 커뮤니케이션에서 출발해, 스트레스 상황에서의 인지 처리(Cognitive)·적응 반응(Adaptive)이 발화 방식(Paralinguistic)에 어떻게 반영되는지를 데이터로 구조화하는 엔진으로 발전해왔습니다.

  1. 2015.10

    LOZEE Prototype

    대화 데이터를 기반으로 감정·어조·표현력·관계 감정을 시각화하는 리포트 개념을 구축했습니다. (초기 문제정의: 감정과 발화의 연결)

    Conversation-based Report Emotion Visualization
  2. 2025

    Paralinguistic Focus 전환

    대화 의미(텍스트) 중심에서, 말속도 급변·떨림·멈춤·리듬 붕괴 등 비언어 음성 지표(Paralinguistic features) 분석으로 중심축을 이동했습니다.

    Speech Features Non-verbal Signals Session-level Analysis
  3. 2025.12

    CASP Engine v1.0

    파라링귀스틱 지표 추출과 스코어링을 엔진화하여, 다양한 제품/환경에 재사용 가능한 분석 코어로 확장합니다.

    Engine Core Indexing Integration-ready SDK / API
  4. 2026.01

    CASP Protocol v1

    데이터 수집 → 품질 판단 → 지표 산출 → 해석/피드백의 흐름을 재현 가능한 표준 프로토콜로 정리했습니다.

    Protocolization Quality Gate Scoring
  5. 2026.06 예정

    CASP Protocol v2

    CASP Protocol v2는 Cognitive–Adaptive Stress Paralinguistic 관점을 유지하면서, 다회 세션·실환경 데이터를 전제로 한 운영·확장 중심 프로토콜입니다. 단일 측정이 아닌 시간에 따른 변화(trajectory)와 환경/과제 맥락을 함께 해석하도록 설계되었습니다.

    Longitudinal Context-aware Protocol v2

CASP Protocol v1

CASP Protocol v1은 세션 기반 음성 데이터를 반복 측정 가능한 지표로 전환하기 위한 표준 흐름(수집→품질→지표)을 정의했습니다. SQ 지수는 개인의 시스템화 경향을 나타내는 기준축으로 활용되어, 음성 지표와의 관계를 검증 가능한 형태로 정리했습니다.

Input

세션 음성 + 설문(SQ 등) 결합

Quality Gate

SNR/길이/말-침묵 비율 기반 품질 판정

Indexing

말속도·멈춤·리듬·변동성 지표 산출 및 인덱싱

CASP Protocol v2

v2는 실환경에서의 반복 사용을 전제로, 다회 세션 변화(trajectory)와 맥락(context)을 포함해 해석하도록 확장했습니다. 또한 제품/플랫폼 적용을 위해 SDK/API 연동을 고려한 운영 구조를 갖춥니다.

Longitudinal

단일 측정이 아닌 세션 간 변화 추적

Context-aware

과제/환경/관계 맥락과 함께 기록·해석

Integration

Engine 및 SDK/API 기반 연동 고려

연구 협업 · PoC · 데이터 프로젝트

기관/기업 환경에 맞춘 프로토콜 적용 및 검증 협업이 가능합니다.
